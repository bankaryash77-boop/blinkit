Google Sheet Integration Guide

Control your shop from Google Sheets

Change any product price, hide a product, add a sale badge — all by editing your Google Sheet. No coding needed. Changes reflect on your website within seconds of the next page load.

1
Import the template into Google Sheets

The zip file includes a file called purefarm_products_sheet.csv with all 19 products pre-filled. Import it like this:

1. Go to sheets.google.com → click Blank spreadsheet

2. Go to File → Import → Upload

3. Upload purefarm_products_sheet.csv

4. Choose "Replace spreadsheet" and "Comma" as separator → click Import

5. Rename the tab at the bottom to products

⚠️ Make sure Row 1 stays as headers: id, name, price, oldPrice, badge, badgeType, weight, category, active — do not delete or rename these.
2
Publish the sheet as CSV

1. Click File → Share → Publish to web

2. In the first dropdown select your products tab (not "Entire Document")

3. In the second dropdown select Comma-separated values (.csv)

4. Click Publish → click OK

5. Copy the URL that appears — it will look like this:

https://docs.google.com/spreadsheets/d/e/2PACX-1v.../pub?gid=0&single=true&output=csv
✓ Once published, Google automatically updates the CSV every time you save the sheet. You don't need to republish after changes.
3
Paste the URL into common.js

Open js/common.js in any text editor (Notepad, VS Code etc.) and find line 8:

const SHEETS_CSV_URL = ''; // paste your Google Sheet CSV URL here

Replace the empty quotes with your URL:

const SHEETS_CSV_URL = 'https://docs.google.com/spreadsheets/d/e/YOUR_ID/pub?gid=0&single=true&output=csv';

Save the file. That's it — the connection is live.

Column reference — what each column does
Column What it controls Example values
id Matches product in the code. Never change this. 1, 2, 3 … 19 REQUIRED
name Product display name shown on card Fresh Tomatoes optional
price Current selling price in ₹ (number only, no ₹ symbol) 40, 120, 280 REQUIRED
oldPrice Original/crossed-out price. Leave empty if no discount. 50, 140 or leave blank optional
badge Text shown on the product badge/label 16% OFF, Fresh, Organic, Best Seller, New optional
badgeType Controls badge colour. Must match one of the values below. disc, fresh, organic, bestseller, new optional
weight Shown under product name (like Blinkit) 500 g, 1 kg, 200 ml, 1 piece optional
category Which sidebar category the product appears in vegetables, fruits, dairy, bakery optional
active Show or hide the product on the website TRUE or FALSE REQUIRED
🏷️
badgeType colour guide

The badgeType column controls the badge colour. Use exactly these values:

disc
Purple badge · for % OFF discounts
fresh
Blue badge · for Fresh / Daily items
organic
Green badge · for Organic products
bestseller
Yellow badge · for Best Seller
new
Light green badge · for New arrivals

If you leave badge empty, no badge appears at all.

✏️
Common tasks — what to do in the sheet

Change a price: Just type the new number in the price column. Save. Done.

Add a sale / discount: Set oldPrice to the original price, set price to the new lower price, set badge to "20% OFF" (or calculate the %) and badgeType to "disc".

Remove a discount: Clear oldPrice, clear badge and badgeType.

Hide a product (out of season / out of stock): Change active to FALSE. Product disappears from the site.

Show it again: Change active back to TRUE.

Mark as Best Seller: Set badge to "Best Seller" and badgeType to "bestseller".

Change weight shown on card: Edit the weight column e.g. "250 g" or "2 kg".

⚠️ After every change, wait ~1 minute for Google to update the published CSV. Then open your website — the first load may show old data for 2-3 seconds while the sheet is fetched. From the second load onwards it's instant.
➕
Adding a completely new product

New products added only in the sheet won't have images (since images are in the code). For full new products with images, you need to:

Option A — Quick (no image): Add a new row in the sheet with a new id (e.g. 20), fill all columns. Product will appear with a broken image or placeholder. Good for testing prices.

Option B — Full (with image): Add the product image to the imgs/ folder, add the product entry to PRODUCT_CATALOGUE in common.js, then control price/active from the sheet as normal.

💡 For most day-to-day tasks (price changes, hiding products, adding badges) you only ever need to touch the Google Sheet — never the code.
📱
Other things to update in common.js

Open js/common.js and update these two lines at the very top:

const WHATSAPP_NUMBER = '919373828887'; // ← your number with country code, no + or spaces const SHOP_NAME = 'PureFarm.'; // ← your shop name (appears in WhatsApp messages)

Replace 919373828887 with your own WhatsApp number. Format: 91 followed by 10-digit number, no spaces, no +.

Example: if your number is 98765 43210 → write 919876543210
